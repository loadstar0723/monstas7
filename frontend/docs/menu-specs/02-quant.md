## 1. 백테스팅 (/quant/backtesting)
### 핵심 기능
- **전략 시뮬레이션**
  - 10년 이상 히스토리컬 데이터

  - 슬리피지/수수료 반영

- **성과 분석**
  - 샤프 비율
  - 최대 낙폭 (MDD)
  - 칼마 비율
  - 소르티노 비율
  - 승률/손익비

### UI/UX 요소
- 비주얼 백테스팅 빌더
- 드래그앤드롭 전략 조합
- 실시간 성과 차트
- 몬테카를로 시뮬레이션

### 전략 빌더

- 파이썬 스크립트 지원
- 백테스팅 통합
- 실시간 실행

### 리스크 관리
- 포지션 사이징
- 손절/익절 자동화
- 최대 노출 제한
- 상관관계 분석

### 성과 분석
- 실시간 P&L
- 드로우다운 분석
- 리스크 조정 수익률
- 벤치마크 비교

### 구독 등급별 차별화
- **Silver**: 기본 백테스팅, 3개 전략
- **Gold**: 전체 백테스팅, 10개 전략
- **Platinum**: ML 전략, 무제한 전략
- **Diamond**: 옵션 전략, API 액세스
- **Black**: 커스텀 전략 개발 지원
