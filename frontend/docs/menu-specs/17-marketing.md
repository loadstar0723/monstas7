  - 판매 도구

  - 리드 공유

### 대리점 지원
- **영업 지원**
  - 영업 키트
  - 데모 계정
  - 교육 프로그램
  - 인센티브

### B2B 마케팅
- **기업 고객**
  - 사례 연구
  - 백서
  - ROI 계산기
  - 엔터프라이즈 데모

### 구독 등급별 마케팅 도구
- **Silver**: 기본 추천 링크
- **Gold**: 이메일 마케팅, 추천인 대시보드
- **Platinum**: 소셜 미디어 도구, 제휴 프로그램
- **Diamond**: 모든 마케팅 도구, API
- **Black**: 전담 마케팅 매니저, 공동 마케팅
