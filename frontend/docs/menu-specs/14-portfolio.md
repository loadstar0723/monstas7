  - 자산별 비중
  - 24시간 변화

  - 거래소별 분산

### 구독 등급별 차별화

- **Platinum**: 세금 리포트, AI 조언
- **Diamond**: 모든 기능 + API
- **Black**: 전문가 포트폴리오 관리
