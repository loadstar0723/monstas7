# 👥 회원 관리 시스템 - 상세 기획서

## 1. 회원 목록 (/members/list)
### 핵심 기능
- **회원 데이터베이스**
  - 전체 회원 조회
  - 역할별 필터링 (본사/총판/대리점/구독자)
  - 구독 티어별 분류
  - 활성/비활성 상태

- **검색 및 필터**
  - 이름/이메일 검색
  - 가입일 범위
  - 최종 접속일
  - 구독 상태

### UI/UX 요소
- 데이터 테이블
- 고급 필터
- 일괄 선택
- 엑셀 내보내기

### 권한 설정
- 본사: 전체 조회/수정
- 총판: 산하 회원만
- 대리점: 직접 가입자만
